# Secure Financial Transaction System

## Overview
This project demonstrates a secure financial transaction system built using Flask and SQLite with cybersecurity-focused controls.

The project simulates:
- JWT authentication
- Password hashing with bcrypt
- Multi-factor authentication (OTP)
- SQL injection protection
- Web Application Firewall (WAF)
- HTTPS/TLS encryption
- Network traffic analysis using Wireshark
- Audit logging

## Features
- User registration/login
- Secure payment processing
- High-value transaction approval workflow
- TLS encrypted communication
- SQL injection detection
- Audit event tracking

## Technologies Used
- Python
- Flask
- SQLite
- bcrypt
- JWT
- pyotp
- OpenSSL
- tcpdump
- Wireshark

## Security Controls
- JWT authentication
- bcrypt password hashing
- OTP verification
- Basic WAF filtering
- TLS encryption
- Audit logging

## Attack Simulations
- SQL injection attempts
- HTTP token interception
- HTTPS encrypted traffic analysis
